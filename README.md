# 🎬 The Film Oracle

**The Film Oracle** is an intelligent movie exploration platform designed to transform how audiences discover cinema.

Instead of endlessly scrolling, users explore films through interactive tools, curated categories, mood-based matching, and global market discovery — all within one unified experience.

🔗 **Live Demo:**  
https://sushanth-hebri.github.io/The-Film-Oracle/

---

## 🚀 What Makes It Different?

The Film Oracle is not just a movie listing site.  
It is a **multi-dimensional discovery engine** built to make film exploration engaging, intuitive, and interactive.

It blends structured movie data with personalized tools and gamified discovery features to create a smarter way to find what to watch.

---

## ✨ Loaded Features

- Trending movies with rating, genre, cast, and release year  
- Global language-based movie filtering  
- Cast-based search with multi-cast combination support  
- Personal movie rating system  
- Head-to-head movie comparison tool  
- Personal watchlist creation and management  
- Director’s Cut – curated hidden gems collection  
- Random movie recommendation (Oracle Spin)  
- Decade-wise movie exploration  
- Mood-based vibe filter  
- Top-rated movies leaderboard  

---

## 🛠 Tech Stack

- HTML5  
- CSS3

## 🌍 API Used

This project uses movie data provided by **The Movie Database (TMDB) API**.

## 🎯 Vision

The Film Oracle aims to evolve into a comprehensive movie intelligence platform — combining discovery, personalization, and interactive exploration into a single cinematic ecosystem.
